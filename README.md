# Student BST — Student Records System

A menu-driven C++ console app that manages student records using a Binary Search Tree. Supports insert, search, and sorted display via in-order traversal.

## Files
## Features
- Add student (ID, name, department)
- Search student by ID
- Display all students sorted by ID
- Duplicate and invalid ID prevention

## Run
```bash
g++ main.cpp studentBST.cpp -o studentBST
./studentBST
```

## Concepts Used
C++ · Binary Search Tree · Recursion · OOP · In-order Traversal · Dynamic Memory Allocation
